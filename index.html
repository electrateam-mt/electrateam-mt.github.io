What I do

Algorithm Design

In the first step, I will design a comprehensive algorithm to guide the project. This design process will assist in identifying the suitable microcontroller and essential electronic components. By outlining a systematic plan, we'll ensure a logical progression, enabling informed choices for the project's components. This thoughtfully designed algorithm establishes a strong framework for project execution, setting the stage for a productive and streamlined development process.

Hardware Design

In my hardware design work, I utilize Altium software for efficient and reliable designs. I carefully consider component placement, signal integrity, power and ground planes, and EMI/EMC compliance. Altium allows me to optimize signal paths through strategic component placement, impedance control, and length matching. I establish robust power and ground planes and address EMI/EMC concerns using shielding and filtering techniques. With Altium, I create high-performance PCB designs that meet modern electronics' demands.

Embedded Programming

During the embedded coding phase, I employ versatile frameworks such as ARM, esp32, and Arduino. The selection is tailored to each project's unique demands, aligning with technical needs, financial considerations, and overall objectives. This flexibility allows me to make informed choices while utilizing C and C++ languages to craft efficient and effective code solutions.

Filter Design

In certain projects, the device's task involves receiving and processing data from a sensor. To extract the desired data signal from other signals, such as noise, I design digital filters using MATLAB software. These filters are then implemented as programming code in the microcontroller. This enables real-time filtering of the data signals, allowing for further processing and analysis.
